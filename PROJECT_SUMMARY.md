## ✅ 완료된 핵심 기능

### 1. 자동 뉴스 크롤링
- TVING 뉴스 사이트에서 기사 자동 수집
- 브루트포스 방식 ID 탐색 (일일 200-500개 기사)
- 마지막 확인 ID 자동 저장

### 2. 이메일 뉴스레터
- Gmail SMTP 연동
- 반응형 HTML 템플릿
- 매일 오전 7:30 자동 발송 (node-cron)

### 3. 구독자 관리
- SQLite 데이터베이스
- AES-256-CBC 이메일 암호화
- 구독/삭제 기능

### 4. 관리자 페이지
- 구독자 목록 조회
- 개별/전체 테스트 발송
- 발송 내역 조회

## 🏗️ 기술 스택

```
Frontend:  HTML5 + CSS + JavaScript
Backend:   Node.js + Express
Database:  SQLite (sql.js)
Email:     Nodemailer (Gmail SMTP)
Scheduler: node-cron
Deployment: GCP Compute Engine (준비 완료)
```
